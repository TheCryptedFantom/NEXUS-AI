# NEXUS-AI

To use this AI, you need to have three API keys.
# Groq
Quick Start guide: https://console.groq.com/docs/quickstart                                                                                                                                             
Variable in .env file = "GROQ_API_KEY"

# Porcupine Wake Word
https://picovoice.ai/docs/api/porcupine-python/                                                                                                                                            
Variable in .env file = "ACCESS_KEY"

# Cartesia
https://play.cartesia.ai/keys                                                                                                                                                                          
Variable in .env file = CARTESIA_API_KEY
